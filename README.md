# 3.16日志
## Tree（决策树与随机树）
当下完成了决策树与随机树的建模和分析，正确率：随机树0.5467，决策树0.4424。目前的数据来看
1，数据极其容易将吸毒归类为未吸毒。
2，随机森林全面优于决策树，Test Macro F1 高出 0.10，原因是多棵树投票比单棵树更稳健，抗过拟合能力更强
3，两个模型共同的瓶颈是 Decade Ago 的 F1 只有 0.34，仅 210 个样本且人格特征与其他类高度重叠，class_weight 补偿效果有限
4，决策树深度只需 5 层说明数据本身的可分性较低，更深的树也无法带来更好的结果
5，两个模型的 CV 和 Test Macro F1 接近，说明没有过拟合，泛化能力正常

# 3月19号日志
## 神经网络NN与KNN邻近
当下完成了 KNN 与两阶段神经网络的建模和分析，准确率：两阶段神经网络 0.72，KNN 0.72。目前数据来看：

1，两个模型准确率相同，但 macro F1 差距明显，两阶段神经网络（0.51）显著优于 KNN（0.43），说明单看 accuracy 会误导结论。
2，KNN 对 Decade Ago 的识别几乎失效（recall 0.07），原因是该类样本量极少（42个），在高维特征空间中与其他类高度重叠，最近邻投票被多数类压制。
3，两阶段神经网络的核心优势在于将三分类拆解为两个二分类，第一阶段专注识别 Never Used，有效避免了多类别相互干扰。
4，两阶段方法的固有局限是第一阶段的误判无法在第二阶段弥补，导致 Decade Ago 和 Recent 的最终 recall 仍然偏低。
5，综合来看，两阶段神经网络是四个模型中表现最均衡的，推荐作为最终模型。
