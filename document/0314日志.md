## 模块1 --重采样
数据存在严重不平衡问题，上游工作合并少数类后3类占比约7:1:2, 写了一个函数对原表格按类比随机抽样本（默认每类200个，有重复），仅支持pd.dataframe格式，能解决样本比重不均的问题，不过因为是有放回的抽样，所以有重复样本，所以测试集里可能有训练集出现过的样本，也就是说有数据泄露问题，准确率可能虚高

## 模块2 神经网络
结构: $11*64$ -> $ReLU$ -> $64*32$ -> ->$ReLU$ -> $32*3$ 

```python
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.net = nn.Sequential(
            nn.Linear(n_features, 64),
            nn.ReLU(),
            nn.Linear(64, 32),
            nn.ReLU(),
            nn.Linear(32, 3)   
        )
    def forward(self, x):
        return self.net(x)
```
